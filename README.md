# Cogito

*Cogito, ergo sum.* I think, therefore I am.

A macOS PDF reader built to help you think. Most PDF readers are good at displaying pages. Cogito turns reading into thinking.

## Roadmap

See [FEATURES.md](FEATURES.md).

## Building

```
make build
make bundle && make run
```

Requires macOS 14 and Xcode 15.
